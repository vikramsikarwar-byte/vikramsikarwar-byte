### About me

I administer AWS infrastructure across 20+ production accounts —
provisioning, IAM, cost optimization, and security auditing are day-to-day
work, not theory. I've found and fixed real waste
(idle load balancers, orphaned resources, unrotated access keys) and built
internal tooling to keep infrastructure visible and accountable.

Alongside that, I build and ship infrastructure-as-code and CI/CD pipelines
on my own time — Terraform for reproducible AWS environments, GitHub Actions
for automated build/test/deploy, Docker for consistent app packaging, and
Kubernetes for container orchestration. Every repo below is actually
provisioned/run/tested, not just written — see each README for how.

**Currently available for freelance work:** AWS infrastructure setup · CI/CD
pipelines · Dockerizing applications · Terraform IaC · AWS cost & security
audits · Linux server deployment
